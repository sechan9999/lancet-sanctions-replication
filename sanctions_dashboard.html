논문이 실제로 제시한 중심 추정치는 연 56만 명(50년 누적 약 2,800만 명)입니다. 그러나 일부 인터뷰에서 언급된 "3,800만 명"은 이 중심값이 아니라, 95% 신뢰구간의 상한(연 76만 명)을 50년으로 단순히 곱한 값과 사실상 정확히 일치합니다. 이는 탭 05의 독립적인 몬테카를로 재현으로도 동일하게 확인됩니다.

회귀분석 재현 — 이중고정효과(TWFE) 패널 모형

국가·연도 고정효과를 통제한 패널회귀로, 제재가 연령대별 로그 사망률에 미치는 영향을 추정합니다. 논문의 두 가지 사양(비가중·엔트로피 가중)을 모두 재현했습니다.

코드 — 국가·연도 고정효과 패널회귀

mit = γ·Sanctionsit + β·Xit + ηi(국가효과) + δt(연도효과) + εit

코드 보기 / 숨기기
# 논문 부록 식(5) 재현: PanelOLS 이중고정효과 회귀
import pandas as pd
from linearmodels.panel import PanelOLS

df = pd.read_stata('mortality1_r-1.dta')
df = df[(df['year'] >= 1971) & (df['year'] <= 2021)]
df = df.set_index(['countrycode', 'year'])

controls = ['ln_dependency', 'ln_rural', 'lgdppcPWT',
            'ln_eysf', 'polity2', 'war']

for outcome in ['ln_mr_wb_neo_t', 'ln_mr_wb_inf_t', ...]:
    sub = df[['uni_sanctions', outcome] + controls].dropna()
    y, X = sub[outcome], sub[['uni_sanctions'] + controls]
    model = PanelOLS(y, X, entity_effects=True, time_effects=True)
    result = model.fit(cov_type='clustered', cluster_entity=True)
    # -> result.params['uni_sanctions'], result.std_errors['uni_sanctions']
분석 의미: entity_effects=True, time_effects=True가 핵심입니다. 국가 고정효과는 지리·문화 등 "그 나라라서 원래 다른" 부분을, 연도 고정효과는 의료기술 발전 등 "전세계가 같이 겪는 변화"를 걷어내, 순수하게 "제재를 받은 해 vs 안 받은 해"의 차이만 남깁니다. 표준오차는 국가 단위로 클러스터링해, 같은 나라의 여러 해 관측치가 서로 독립이 아니라는 점을 반영합니다.

해석 시 주의할 점: 사망자 수 상위 국가에 중국·인도네시아·파키스탄처럼 인구 대국이 올라오는 것은 "제재가 특히 가혹해서"가 아니라, 같은 비율의 사망률 증가라도 인구가 많으면 절대 인원이 커지기 때문입니다. 실제로 논문 저자들도 부록에서 "이질적인 제재 강도 때문에 국가별 추정치를 제시하지 않는다"고 명시하고 있습니다. 이 차트는 어디까지나 예시적 참고용이며, 제재의 '가혹함'을 나라별로 순위 매기는 지표로 오독해서는 안 됩니다.

# pop_t, dea_t를 0-1 / 1-5 / 5-10 / 10-15 / 15-60 / 60-80 구간으로 합산

# 반사실적(counterfactual) 사망률: 논문 식(14)
# mr_hat = mr / exp(beta)  ->  초과사망자 = 관측사망자 * (1 - exp(-beta))
merged['excess_deaths'] = np.where(
    merged['uni_sanctions'] == 1,
    merged['dea_t'] * 1000 * (1 - np.exp(-merged['beta'])),
    0
)
annual_deaths = merged[merged.year.between(2012,2021)].groupby('bin')['excess_deaths'].sum() / 10
분석 의미: "반사실적 사망률"이란 "만약 이 나라가 제재를 받지 않았다면 있었을 사망률"을 뜻합니다. 회귀계수 β만큼 로그 사망률이 높아졌다고 추정되므로, 그 효과를 역산(÷exp(β))해 "제재가 없었을 경우"를 복원하고, 실제 관측된 사망자 수와의 차이를 "초과사망자"로 정의합니다.

몬테카를로 검증 — "3,800만 명" 주장의 실체

회귀계수의 표준오차를 반영해 1,000회 시뮬레이션을 돌리고, 그 분포에서 신뢰구간을 도출한 뒤 논문 공식 수치 및 세간의 "3,800만 명" 주장과 비교합니다.

코드 — 몬테카를로 시뮬레이션 (N=1,000)

코드 보기 / 숨기기
# 논문 부록 방법 재현: 계수의 정규분포에서 반복 추출
np.random.seed(42)
N_SIM = 1000
sim_totals = np.zeros(N_SIM)

for s in range(N_SIM):
    total = 0.0
    for age_bin in bins:
        # 논문과 동일: beta ~ Normal(평균, 표준오차)에서 추출
        beta_draw = np.random.normal(loc=coefs[age_bin], scale=ses[age_bin])
        excess = dea_by_bin[age_bin] * (1 - np.exp(-beta_draw))
        total += excess.sum()
    sim_totals[s] = total / 10  # 10년 평균 -> 연율화

mean_est = sim_totals.mean()
ci_low, ci_high = np.percentile(sim_totals, [2.5, 97.5])

# '3800만명' 검증: 신뢰구간 상한을 50년으로 단순 외삽
extrapolated = ci_high * 50
분석 의미: 회귀계수는 하나의 확정된 숫자가 아니라 불확실성을 가진 확률분포입니다. 표준오차를 반영해 1,000개의 "있을 법한 세계"를 시뮬레이션하면, 단일 중심값 대신 전체 사망자 수의 분포와 신뢰구간을 얻을 수 있습니다. 이 분포의 양쪽 끝(꼬리)은 통계적으로 가능은 하지만 "가장 그럴듯한 값"은 아닙니다.
